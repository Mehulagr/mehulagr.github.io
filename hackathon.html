    Mobile app for Web-opac

    For every course a list of course book and reference books is provided. And it is a tradition to rush to library to get the book first. This process involves getting the list of books from the intranet and getting that printed. Followed by searching it on computer of the library and getting its location on library. Followed by rushing towards it to get it first. We tried to improve this process along with the process of getting any other book in library by providing mobile app that can be used to access the web-opac, which has pre-loaded data of your course books depending on your roll-no.


    IIT Guwahati Webmail Interface Redesign

    While working with a computer science student of IIT Guwahati for Chrome Extension which would imply new user-interface on IITG webmail on adding it to your Google chrome, we did UX design part of it. Problems which we found in existing UI were: -
    1. Important features which are supposed to be frequently used were nearly hidden in different options in fact some students were unaware about some features that they are there in webmail.
    2. Visuals. Although one can customize its colors but again this feature is not upfront and also basic theme itself is not nice.
    3. Titles or labels of some options are not intuitive.
    4. Grouping of content.

    Therefore, we studied other e-mail systems UI like Gmail and Yahoo. We decided on content that is to be displayed on main page. Then we decided how it would be shown – grouping and placement of buttons etc. Finally we designed mock-ups for each individual page. We planned for two colour themes initially – Dark and Light.

    IIT Guwahati Hackathon

    This is a work done by us for overnight Hackothlon in Kriti, inter- hostel technical competition of IIT Guwahati.
